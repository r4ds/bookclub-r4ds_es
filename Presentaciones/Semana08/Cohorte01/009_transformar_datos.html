Manejar datos

En (Grolemund and Wickham 2019, Cap 9) aprenderás a manipular datos. Es decir:

  • Importarlos
  • Ordenarlos
  • Transformarlos

Manejar datos

  • Para manejar los datos se introducirá el objeto denominado tibble
  • También conocerás cómo traer los datos al disco
  • Finalmente entederás el concepto de datos ordenados

También para manipular datos necesitaras conocer acerca de:

  • Datos relacionales
  • Cadenas de caracteres
  • Factores
  • Fechas

Referencias

Grolemund, Garrett, and Hadley Wickham. 2019. R Para Ciencia de Datos. http://es.r4ds.hadley.nz/.
